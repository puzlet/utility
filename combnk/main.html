combnk

Given non-negative integers k and n, generate all size p combinations of the integers from 0 to n-1 in sorted order (each combination is sorted and the entire table is sorted). For example, comb(5,3) is 0 1 2, 0 1 3, 0 1 4, 0 2 3, 0 2 4, 0 3 4, 1 2 3, 1 2 4, 1 3 4, 2 3 4 See: [http://rosettacode.org/wiki/Combinations#CoffeeScript, http://rosettacode.org/wiki/Combinations#CoffeeScript] [http://octave.sourceforge.net/statistics/function/combnk.html, http://octave.sourceforge.net/statistics/function/combnk.html]
